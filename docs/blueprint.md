# **App Name**: Kinefonia Studio

## Core Features:

- Camera Input: Capture video input from the user's camera.
- Visual Modes: Apply different visual modes to the camera input, such as Sonic Mesh, Kaleido, Fractal, Trace, Slitscan, Particles, Heatmap, Network, Scanner, and Poetic. Each mode will have adjustable parameters.
- Parameter Controls: Provide UI sliders to adjust parameters specific to each visual mode, such as tension, perspective, amplitude, segments, zoom, rotation speed, etc.
- Camera Filters: Apply different camera filters to the video feed, such as Real, BW Contrast, Vibrant, Dream, and Invert.
- Audio Visualization: Analyze audio input and map it to visual elements in real-time.
- Poetic Generation: Use a generative AI tool to create short, cryptic phrases based on the current motion and audio levels, displaying the output in the poetic mode.
- Presets: Allow the user to save and load parameter presets, enabling quick switching between configurations.

## Style Guidelines:

- Primary color: Deep magenta (#FF0055) for a cyberpunk, edgy feel.
- Background color: Near-black (#0A0A0F) for high contrast and to emphasize the visuals.
- Accent color: Electric cyan (#00E5FF) for interactive elements and highlights.
- Headline font: 'Montserrat' (sans-serif) for headlines, strong and modern.
- Body text font: 'Courier Prime' (monospace) for code-like and technical feel.
- Use 'Lucide React' icons for a consistent, modern, and clean UI.
- Split the screen into a left panel for the video output and a right panel for controls.
